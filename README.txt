QUICK-START SETUP GUIDE:
========================================================================
1. Extract the zip file to a new folder on your computer.
2. Open your terminal in that extracted directory.
3. Run the following commands:
   
   npm install        <-- Downloads and configures local dependencies
   npm run dev        <-- Starts the local Vite development server
   npm run build      <-- Compiles static assets into the 'dist/' folder for production
